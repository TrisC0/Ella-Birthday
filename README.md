# 🎂 Happy 15th Birthday, Ella! 🚵‍♀️🐹🎵

A special, hand-made **animated pixel-art birthday card** for Ella's 15th —
built to be hosted for free on GitHub Pages.

It features all of Ella's favourite things:

- 🚵 An **enduro mountain biker** catching air over a mountain trail
- 🐹 Her **grey hamster** as the party mascot (in a tiny party hat, holding a balloon)
- 🎵 Floating **Billie**-style music notes
- 🎨 A bold **blue / teal / green** "boy colours" palette
- 🎂 A birthday cake with candles, twinkling stars, a crescent moon, bunting, and lots of confetti
- ✨ Big animated lettering spelling **ELLA**

Everything is true pixel art, drawn pixel-by-pixel on an HTML5 canvas — no images,
no libraries, just one self-contained `index.html`.

## ▶️ View it

Just open `index.html` in any web browser.

Two buttons let you:
- 🎉 **More Confetti** — fire a confetti burst
- 🎵 **Play Tune** — play a little "Happy Birthday" melody

## 🌍 Host it for free on GitHub Pages

1. Push this repo to GitHub (public).
2. Go to **Settings → Pages**.
3. Under **Build and deployment → Source**, choose **Deploy from a branch**.
4. Pick the branch (e.g. `main`) and the `/ (root)` folder, then **Save**.
5. After a minute, your card will be live at:
   `https://<your-username>.github.io/<repo-name>/`

Share that link with Ella on her birthday! 🎁
